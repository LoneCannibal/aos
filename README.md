# AOS Project: Distributed Ticket Booking System
AOS project made by Team No. 10  
Collaborators: Aakash S D, Bhagavath M S, Harshvardhan Kumain

## How to use
1. Install dependencies 
## ```python -r requirements.txt```

2. Generate proto python files  
For Windows run
## ```protoscripts.bat``` 
 
For Linux run
## ```protoscripts.sh```

3. Run the client, server and llm python scripts  inside client, server and llm-server directories
